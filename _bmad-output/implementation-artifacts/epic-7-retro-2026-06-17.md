## What Went Well

1. **Foundation-first order paid off.** 7.2 (`pageInsets`, tokens) unblocked 7.1 results spacing, 7.3 back offset, and 7.4 top bar insets without rework.
2. **Shared components reduced duplication.** `DiscoveryTopBar`, `OverlayBackButton`, `PageContent` replaced one-off patterns across Home/Explore/Info/Quiz/Legal/Detail.
3. **Content + code split for 7.5 and 7.6.** Editorial passes (chip shortening, `around_area`) paired with code guardrails (`CHIP_MAX_LENGTH`, `deriveAroundArea`).
4. **PO feedback loops were fast.** 7.4 refined after CR (Home search icon vs inline, Info top bar, spacing, X-button race fix) without reopening the whole story.
5. **Targeted test scripts.** `verify-around-area.ts` gives cheap regression coverage for 7.6 fallback logic.

---

## Challenges & Growth Areas

1. **7.1 animation freeze.** Initial `animationend`-based slides caused a stuck state; timeout-based advance was the durable fix — prefer integration tests or manual SE checklist for animation stories.
2. **7.4 pointer-event races.** Home search X button close/reopen race required `data-discovery-search-toggle` + edge-triggered analytics — document pattern for overlay + document listeners.
3. **Visual spacing ≠ token spacing.** Top bar “equal margins” needed asymmetric `pl-top-bar-x` / `pr-screen-x` because hamburger touch target skews perception.
4. **7.5 display vs source length.** `num_runs` YAML can stay long while chip shows `"14 runs"` — CR caught tooltip/value distinction; content editors need the distinction documented.
5. **Epic 7 manual QA is additive.** Launch checklist §4 predates Wave 2; separate sign-off doc avoids conflating pre/post-launch passes.

---

## Key Insights

1. **Post-launch epics should ship their own QA sheet.** Wave 2 touches quiz, global chrome, and all detail pages — one checklist row in §4 is insufficient.
2. **Airport ≠ area for editorial headings.** `gateway_airport` belongs in transport copy only; `around_area` prevents recurring PO confusion.
3. **Chip rows need three layers:** shorten YAML, demote long values to facts, UI wrap/truncate — any single layer alone fails on worst offenders.
4. **Defer PO deviations in story artifacts.** 7.4 artifact updated post-refinement (Home icon layout, Info scope) — keeps CR and sprint history honest.
