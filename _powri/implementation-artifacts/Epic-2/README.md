# Epic 2: Discover Your Resort

Build the core discovery experience: first-time onboarding, home hero, resort cards for all 20 resorts, filters, inline search, the Explore tab, and the Find My Resort quiz with results podium. This epic delivers the primary user journey for finding a ski resort.

## Stories

- **2.1** — First-Time Onboarding Flow
- **2.2** — Home Hero Carousel & Tagline
- **2.2.1** — Compact Home Hero Height (30vh)
- **2.3** — Resort List & Resort Cards (All 20)
- **2.4** — Filter Chips & Instant List Filtering
- **2.5** — Inline Search by Resort Name
- **2.6** — Explore Tab
- **2.7** — Find My Resort Quiz
- **2.7.1** — Quiz Results Podium & Cosmic Q4 Typography Fix
