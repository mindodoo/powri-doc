## What Went Well

1. **Parallel story tracks worked.** 8.1 (copy), 8.2 (assets), and 8.3 (motion) ran independently, then 8.4 wired them — matching the epic shard's recommended order without integration thrash.
2. **Scoring guardrail held.** Every story reiterated "do not change `score.ts` enums"; `test:quiz` Ninja persona caught regressions cheaply across all four stories.
3. **PO-driven 8.3 pivot improved UX.** Shifting magic to the intro (fortune artwork + candle path) and keeping question screens clean (white buttons + sparkle) reduced visual noise while preserving Cosmic identity — better than the original per-question candle + twilight buttons spec.
4. **Analytics hygiene.** `cosmicStoryArcAnalytics()` centralises `story_arc_version: "forest-v1c"` — Cosmic-only, no leakage on Serious events; tracking-plan updated in 8.1.
5. **Asset pipeline repeatable.** GenerateImage → sharp WebP compression hit per-file ≤120 KB targets; `content/quiz-illustrations/README.md` records prompts and sizes for future arc bumps.
6. **Scope revisions documented honestly.** 8.3 story artifact, QA doc scope table, and 8.4 integration traceability all reflect shipped UX vs original epic draft — keeps CR and sprint history accurate.

## Challenges & Growth Areas

1. **8.3 diverged from epic shard ACs.** Original spec: colored cosmic buttons + floating candle on every question screen. Shipped: intro-only magic + standard answer chrome. Epic shard and PRD still describe the old UX — only QA doc and story artifacts capture the revision.
2. **Total asset weight grew with intro art.** Four scenes met ≤500 KB; adding `cosmic-fortune-start.webp` (~287 KB) brought total to ~756 KB. LCP on throttled 4G verified in manual QA (8.2.3 pass).
3. **Illustration consistency needed PO loop.** Scene 3 (Nordic stone arch gates) required explicit PO approval; Scenes 1/2/4 spot-check noted as optional — batch AI consistency remains a risk for future arcs.
4. **Manual QA deferred at dev-complete, closed at retro.** 8.4 shipped with PO pass deferred (Epic 7 pattern); Mindodoo confirmed full sign-off 2026-06-18 at retro closure.
5. **Release gate analytics completed post-deploy.** PostHog Cosmic vs Serious `quiz_completed` comparison passed per PO confirmation.

---

## Key Insights

1. **Intro is the right place for Cosmic spectacle.** Question screens benefit from illustrations + sparkle; heavy chrome (candles, colored buttons) on every scene adds clutter without improving comprehension.
2. **Content epics need scope-change breadcrumbs in planning shards.** When PO revises UX mid-epic, update the epic shard (or add a `## Shipped deviations` section) so future agents don't implement the old spec.
3. **Asset budgets should include intro/hero art.** Scene-only caps are insufficient when integration adds standalone intro illustrations.
4. **`COSMIC_STORY_ARC_VERSION` constant enables future arcs.** Bumping `forest-v1c` → `forest-v2` is a copy + asset swap without scoring changes — good separation of concerns.
5. **Serious mode is the regression canary.** Any Cosmic change must prove Serious unchanged via automated persona tests before merge.
