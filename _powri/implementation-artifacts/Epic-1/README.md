# Epic 1: App Foundation & Content Pipeline

Establish the Next.js application baseline, design system, markdown content pipeline, analytics/consent infrastructure, and core app shell. This epic lays the technical foundation that all subsequent Phase 1 features build on.

## Stories

- **1.1** — Initialize Next.js Project & Deployment Baseline
- **1.2** — Design Tokens & Default Theme Typography
- **1.3** — Markdown Content Loader & Resort Types
- **1.4** — App Shell, Routing & Bottom Navigation
- **1.5** — Analytics, Consent Banner & Common Event Properties
- **1.6** — Security Headers & API Route Scaffold
- **1.7** — i18n Scaffold (English Only)
