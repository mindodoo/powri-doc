Phase 2 adaptive navigation (mobile bottom nav + desktop top bar), hamburger overflow menu, save-resort heart for guests and authenticated users, and the saved list page with login merge. Depends on Epic 9.2 for session-aware sync.

- **10.1** — Adaptive Navigation — Mobile Bottom + Desktop Top
- **10.2** — Unified Desktop Nav & Hamburger Overflow
- **10.2.1** — Mobile Page Titles Hidden Under Fixed Top Bar
- **10.2.2** — E2E stale-server false failures + 1px AppMenuSheet gap fix
- **10.3** — Save Resort Heart — Guest & Authenticated
- **10.4** — Saved List Page & Login Merge
